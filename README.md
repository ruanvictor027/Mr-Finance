# MR Finance

Painel de finanças pessoais — aplicação web de página única (HTML/CSS/JS, sem backend).
Importa extratos OFX/CSV/PDF e guarda tudo localmente no navegador (`localStorage`).

## Publicação (GitHub Pages)

Este repositório serve o app direto do `index.html`. Em **Settings → Pages**,
selecione o branch `main` e a pasta `/ (root)`. O site fica disponível em
`https://SEU-USUARIO.github.io/NOME-DO-REPO/`.

## Privacidade

Nenhum dado financeiro é versionado aqui — apenas o aplicativo (`index.html`).
Os dados do usuário ficam apenas no navegador de quem usa o app.
