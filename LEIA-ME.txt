MR FINANCE — BUILD MOBILE (pasta independente)
================================================

Esta pasta contém TUDO que o app mobile precisa para rodar sozinho,
SEM a versão desktop.

ARQUIVOS INCLUÍDOS
------------------
  index.html            -> entrada (vai direto pro app mobile)
  mobile.html           -> o app
  mobile.css            -> estilos
  mobile.js             -> telas/lógica de UI
  mr-core.js            -> núcleo de dados/regras (localStorage)
  manifest.webmanifest  -> PWA (instalável)
  service-worker.js     -> offline/cache (PWA)
  icon.svg              -> ícone vetorial (fallback)

⚠️ FALTAM 2 ARQUIVOS (seu logo em PNG)
--------------------------------------
Coloque AQUI nesta pasta os seus 2 PNGs do logo:
  icon-192.png   (192 x 192)
  icon-512.png   (512 x 512)

Sem eles o app FUNCIONA mesmo assim (cai no icon.svg automaticamente),
mas para o ícone na tela inicial do iPhone/Android ficar com o seu
logo oficial, adicione os 2 PNGs.

COMO PUBLICAR (GitHub Pages)
----------------------------
1. Suba TODOS os arquivos desta pasta (incluindo os 2 PNGs) para o
   repositório.
2. Settings > Pages > Branch: main > pasta / (root) > Save.
3. Abra a URL no celular. Ao instalar ("Adicionar à tela inicial"),
   vira um app.

IMPORTANTE — CACHE DO PWA
-------------------------
Depois de cada atualização, RECARREGUE 2x no celular: o service worker
antigo serve a versão anterior na 1a vez; na 2a o novo assume.

DADOS
-----
Os dados ficam no localStorage DESTE aparelho/navegador. Faça backup
pelas Configurações (Exportar backup) para não perder.
